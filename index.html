Two.js is a two-dimensional drawing api meant for modern web browsers. It is renderer agnostic enabling the same api to draw in multiple contexts: svg, canvas, and webgl.

Overview

  • Scenegraph

    Two.js starts with a scenegraph. This means that when you draw or create an object (a Two.Shape or Two.Group), two actually stores and remembers that. After you make the object you can apply any number of operations onto it â€” e.g: rotation, translation, scale.

  • SVG Interpreter

    Two.js features a Scalable Vector Graphics Interpreter. This means developers and designers alike can create visuals in commercial applications like Adobe Illustrator and bring them into your two.js scene.
